◆ dac_available()

uint8_t dac_available ( uint8_t  num)

See how many samples are still being converted.

Parameters
numThe DAC to use
Warning
This API returns 0xff in case FIFO is full or empty. User need to check the interrupt pending bit to conclude 0 samples in FIFO or 128 samples in FIFO. This function should be called in the I2S interrupt context in case of FT900 Revision B for a reliable count value. For more info, please refer to "TN_159 FT90x Errata Technical Note".
Returns
The number of samples still to be converted, or 0 otherwise
