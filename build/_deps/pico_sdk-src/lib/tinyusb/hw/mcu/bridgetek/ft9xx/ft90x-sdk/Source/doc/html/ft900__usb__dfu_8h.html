Detailed Description

USB DFU class definitions.

This contains USB definitions of constants and structures which are used in USB DFU implementations.

Definitions for USB DFU Devices based on USB Device Firmware Upgrade Specification Revision 1.1 from http://www.usb.org/developers/docs/

enum DFU_STATE

DFU State Codes. Unnamed table page 22. Device state definitions.

Enumerator
DFUSTATE_APPIDLE 

Device is running its normal application.

DFUSTATE_APPDETACH 

Device is running its normal application, has received the DFU_DETACH request, and is waiting for a USB reset.

DFUSTATE_DFUIDLE 

Device is operating in the DFU mode and is waiting for requests.

DFUSTATE_DNLOAD_SYNC 

Device has received a block and is waiting for the host to solicit the status via DFU_GETSTATUS.

DFUSTATE_DNLOAD_BUSY 

Device is programming a control-write block into its nonvolatile memories.

DFUSTATE_DNLOAD_IDLE 

Device is processing a download operation. Expecting DFU_DNLOAD requests.

DFUSTATE_MANIFEST_SYNC 

Device has received the final block of firmware from the host.

DFUSTATE_MANIFEST 

Device is in the Manifestation phase.

DFUSTATE_MANIFEST_WAIT_RESET 

Device has programmed its memories and is waiting for a USB reset or a power on reset.

DFUSTATE_UPLOAD_IDLE 

The device is processing an upload operation. Expecting DFU_UPLOAD requests.

DFUSTATE_DFUERROR 

An error has occurred. Awaiting the DFU_CLRSTATUS request.
