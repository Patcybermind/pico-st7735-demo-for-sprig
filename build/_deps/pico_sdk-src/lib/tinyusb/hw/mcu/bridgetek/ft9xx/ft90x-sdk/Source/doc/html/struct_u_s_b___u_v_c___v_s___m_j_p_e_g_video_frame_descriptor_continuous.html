unsigned char USB_UVC_VS_MJPEGVideoFrameDescriptorContinuous::bmCapabilities

D0: Still image supported. Specifies whether still images are supported at this frame setting. This is only applicable for VS interfaces with an IN video endpoint using Still Image Capture Method 1, and should be set to 0 in all other cases. D1: Fixed frame-rate. Specifies whether the device provides a fixed frame rate on a stream associated with this frame descriptor. Set to 1 if fixed rate is enabled; otherwise, set to 0. D7..2: Reserved, set to 0
