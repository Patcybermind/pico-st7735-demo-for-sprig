Detailed Description

USB BOMS class definitions.

This contains USB definitions of constants and structures which are used in USB BOMS implementations.

Definitions for USB BOMS Devices based on USB Mass Storage Bulk Only Specification Revision 1.0 from http://www.usb.org/developers/docs/
