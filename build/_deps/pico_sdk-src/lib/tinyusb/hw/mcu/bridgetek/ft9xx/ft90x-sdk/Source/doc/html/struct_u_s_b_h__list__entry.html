Detailed Description

Structure that represents a Queue Entry. This data structure is used for the start of a queue of Queue Element Transfer Descriptors. This typically represents an endpoint in the transfer queue list. NOTE: This structure is an EHCI defined structure and must be located in the USBH memory space.
